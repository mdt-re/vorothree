Expand description

A spatial index based on a uniform grid.

This structure divides the 3D space into a fixed number of bins (voxels). It is generally faster than an octree for uniform distributions and allows for O(1) insertion and update operations, but may be less memory efficient for highly clustered data or very large sparse domains.
