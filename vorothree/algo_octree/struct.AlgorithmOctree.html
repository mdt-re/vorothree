Expand description

A spatial partitioning structure based on an Octree.

This structure recursively subdivides the 3D space into eight octants. It is particularly efficient for non-uniform distributions of points, as it adapts the depth of the tree to the local density of points.

Returns an iterator that yields points in the octree ordered by distance from the query point.

This iterator uses a priority queue to traverse the octree, ensuring that closer points (or octants containing closer points) are visited first.
