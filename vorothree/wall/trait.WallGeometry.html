Expand description

Trait defining the geometry and logic of a wall. Must be Send + Sync to support parallel execution in Tessellation.

fn cut( &self, generator: &[f64; D], callback: &mut dyn FnMut([f64; D], [f64; D]), )

Calculates the clipping plane for a given generator. Returns a tuple (point_on_plane, plane_normal). The normal should point OUT of the valid region (towards the region to be clipped).
